## Launching

### Linux
Install dependencies (`pip install ...`) :

* pillow
* feedparser
* requests
* beautifulsoup4
* python-dateutil

Then`cd scripts` and `python main.py`

### Windows
Install dependencies (`pip install ...`) :

* pillow
* feedparser
* requests
* beautifulsoup4
* python-dateutil

Then `cd scripts` and `python.exe main.py`

  

## Utilization

Manage your Flux RSS in `Settings` > `Feeds...`
Actualize with `Settings` > `Actualize`
